# Proyecto de análisis de datos del Titanic

## Descripción
Este proyecto es un análisis de datos del Titanic, en el que se busca responder a preguntas como:
- ¿Cuántos pasajeros sobrevivieron?
- ¿Cuál fue la distribución de géneros de los pasajeros?
- ¿Cuál fue la distribución de edades de los pasajeros que sobrevivieron?
- ¿De qué clase eran los pasajeros que sobrevivieron?

## Resultados
Los resultados del análisis se encuentran en el archivo `titanic.ipynb`.

## App de Streamlit
Se ha creado una app de Streamlit para visualizar los resultados del análisis y una página con un modelo predictivo de supervivencia de un pasajero ficticio. 
La app se encuentra en el archivo `titanic.py` y se puede ejecutar con el comando `streamlit run titanic.py`.
